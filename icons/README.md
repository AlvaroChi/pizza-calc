# icons/

Questa cartella deve contenere le icone PWA:

| File | Dimensione |
|------|------------|
| `icon-192.png` | 192 × 192 px |
| `icon-512.png` | 512 × 512 px |

## Come generarle

Crea un'immagine quadrata 512×512 px con il logo Pizza Tools
(es. emoji 🍕 su sfondo bianco o colorato) e salvala come PNG.

Poi ridimensiona a 192×192 per la seconda icona.

Strumenti gratuiti: [favicon.io](https://favicon.io), [realfavicongenerator.net](https://realfavicongenerator.net)

> Finché le icone non sono presenti la PWA funziona ma non può essere installata
> come app standalone su Android/iOS.
